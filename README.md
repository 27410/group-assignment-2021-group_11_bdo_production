## Project overview

  3. Yields.ipynb: Yield calculation for the two models 
  4. Medium.ipynb: Analysis of different substrates  
  5. Mutations.ipynb: Manual gene knock-outs 
  6. Co_Factor_Swap.ipynb: Co-factor swapping
